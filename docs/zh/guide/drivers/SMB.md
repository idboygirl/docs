# SMB

挂载路径:随便填一个就可以了，如/smb
地址：填SMB的地址
192.168.1.66:445
用户名：smb的账号
密码:smb的密码
分享名称：填smb的网站地址，如//192.168.1.10/smb

其他就默认就可以了


是把smb挂载到Alist，而不是Alist通过smb共享出来，而不是smb能像webdav共享出来。
